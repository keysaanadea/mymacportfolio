Problem Statement

Mengapa BISINDO Penting?

BISINDO adalah bahasa isyarat resmi Indonesia, namun akses terhadap pembelajaran yang mudah dan terjangkau masih sangat terbatas — terutama bagi komunitas tuli dan masyarakat umum yang ingin berkomunikasi secara inklusif.

0.4%
Prevalensi disabilitas pendengaran di Indonesia — sekitar 1.1 juta jiwa (SKI 2023, Kemenkes RI)
4.2%
Prevalensi gangguan pendengaran (semua tingkat) di Indonesia menurut WHO
Active
Research gap — BISINDO recognition masih menjadi topik aktif di IEEE & ScienceDirect (2024–2025), solusi mobile belum banyak

Solusi GESTURA: Menggunakan computer vision dan model CNN untuk mengenali gestur tangan BISINDO secara real-time di kamera ponsel — inferensi berjalan on-device via TFLite tanpa koneksi internet.

Data Pipeline

Dataset & Augmentasi Data

Dataset BISINDO

Dataset berisi gambar gestur tangan untuk alfabet BISINDO. Setiap kelas merepresentasikan satu huruf, dengan variasi dalam pencahayaan, sudut, dan latar belakang untuk ketangguhan model.

Dataset dibagi menjadi dua direktori terpisah: dataset/train/ dan dataset/validation/
26 kelas — satu per huruf alfabet BISINDO (A–Z)
Normalisasi pixel ke [0, 1] via rescale=1/255
Resize gambar ke 224 × 224 sesuai input VGG16

Teknik Augmentasi

Augmentasi via library imgaug — setiap gambar menghasilkan 5 gambar augmented baru.

Fliplr(0.5) — horizontal flip dengan probabilitas 50%
Crop(0–10%) — random crop untuk variasi framing
LinearContrast(0.75–1.5) — penyesuaian kontras
AdditiveGaussianNoise(0–5%×255) — noise
Multiply(0.8–1.2) — variasi kecerahan gambar
Affine(rotate ±20°, scale 0.8–1.2)
Experimentation

Eksperimen Model

Empat arsitektur transfer learning diuji coba dan dibandingkan untuk menemukan model paling optimal dari sisi akurasi, kecepatan inferensi, dan ukuran file.

MobileNet

Arsitektur ringan dirancang untuk mobile — depthwise separable convolution

  • Model sangat kecil, inferensi cepat
  • Ideal untuk resource terbatas
Akurasi lebih rendah untuk dataset kompleks dengan variasi tinggi

VGG16

16 layer deep CNN dengan pola konvolusi 3×3 yang konsisten dan terbukti

  • Performa terbaik pada dataset BISINDO
  • Feature extraction kuat untuk gestur tangan
  • Transfer learning dari ImageNet solid
  • TFLite conversion berhasil

VGG19

Versi lebih dalam dari VGG16 dengan 19 layer

  • Kapasitas representasi lebih besar
  • Cocok untuk dataset sangat besar
Lebih berat, training lebih lama, ukuran model lebih besar

InceptionV3

Inception module paralel — menangkap fitur di berbagai skala

  • Efisien secara komputasi per parameter
  • Multi-scale feature detection
Kompleksitas lebih tinggi, kurang optimal untuk dataset berukuran sedang
VGG16 dipilih sebagai model final karena memberikan kombinasi terbaik antara akurasi klasifikasi gestur BISINDO dan ukuran file yang feasible untuk TFLite deployment.
