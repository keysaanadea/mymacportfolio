Overview

Project Overview

Penelitian ini membangun pipeline klasifikasi berbasis patch untuk membedakan lukisan tradisional Jepang (Ukiyo-e dan Sumi-e) dari lukisan Tiongkok. Tantangan utama terletak pada perbedaan gaya yang sangat halus — sapuan kuas, motif dekoratif, dan penggunaan ruang kosong.


Pendekatan hybrid menggabungkan ResNet50 sebagai feature extractor, SVM sebagai patch scorer, dan hybrid centroid voting sebagai mekanisme keputusan akhir yang dapat dijelaskan.

Patch-Based Approach

Gambar dibagi ke dalam 16 patch untuk klasifikasi granular berbasis lokasi — menangkap karakteristik lokal yang sering terlewat oleh whole-image classification.

Hybrid ResNet50 + SVM

ResNet50 (ImageNet) mengekstrak deep features per patch; SVM dengan kernel RBF menilai probabilitas per patch; centroid voting menentukan prediksi akhir.

Explainability (XAI)

Analisis level patch mengungkap region gambar mana yang mendorong prediksi — memvisualisasikan sapuan kuas dan motif dekoratif yang membedakan kedua gaya.

Model Design

ResNet50 + SVM Hybrid

Pilihan ResNet50 sebagai feature extractor didasarkan pada kemampuannya menangkap hierarki fitur visual dari edge sederhana hingga tekstur kompleks — cocok untuk perbedaan gaya artistik yang halus.


SVM dengan kernel RBF dipilih sebagai classifier karena efektivitasnya di ruang high-dimensional setelah PCA reduction — dan kemampuannya menghasilkan probabilitas per kelas yang diperlukan untuk voting.


ResNet50 di-freeze (ImageNet weights) — hanya layer custom head yang dilatih
Global Average Pooling menghasilkan 2048-dim feature vector per patch
PCA mereduksi dimensi untuk SVM training yang lebih efisien
SVM menghasilkan P(Jepang) dan P(Tiongkok) per setiap patch
Hybrid Voting Mechanism
1. SVM menghitung P(Jepang) per patch
2. Hitung jarak patch ke centroid kelas Jepang & Tiongkok
3. Gabungkan: score akhir = α·P(SVM) + β·sim(centroid)
4. Mayoritas vote dari 16 patch → label gambar
5. Patch dengan score tinggi di-highlight sebagai XAI evidence

Keunggulan hybrid: Prediksi tidak hanya berdasarkan probabilitas statistik (SVM) tapi juga kedekatan geometris dalam feature space (centroid) — lebih robust terhadap patch yang ambigu.

Explainability

XAI — Explainability Berbasis Patch

Salah satu kontribusi utama penelitian ini adalah interpretabilitas visual — model tidak hanya memberikan prediksi, tapi juga menjelaskan mengapa sebuah gambar diklasifikasikan sebagai Jepang atau Tiongkok.


Setiap patch mendapat confidence score. Patch dengan score tertinggi di-overlay pada gambar asli, mengungkap region visual yang paling berpengaruh terhadap keputusan model.


Heatmap overlay menunjukkan patch yang paling "Japanese" atau "Chinese"
Region sapuan kuas halus cenderung dominan untuk Jepang
Region motif kompleks dan pewarnaan kuat untuk Tiongkok
Patch ambigu (score mendekati 0.5) divisualisasikan terpisah

Patch Confidence Map

Setiap 16 patch divisualisasikan dengan warna sesuai confidence score — merah untuk Jepang, biru untuk Tiongkok, abu-abu untuk ambigu

Region Highlighting

Top-5 patch dengan score tertinggi di-overlay pada gambar asli dengan bounding box berwarna — menunjukkan secara visual "di mana" model melihat perbedaan

UMAP Feature Visualization

UMAP mereduksi feature vectors ke 2D untuk memvisualisasikan pemisahan kelas dalam ruang fitur — konfirmasi bahwa kelas memang terpisah secara geometris

Dataset

Dataset & Pengolahan Data

Dataset dikompilasi dari berbagai sumber museum dan repositori seni digital, terdiri dari lukisan tradisional Jepang (Ukiyo-e, Sumi-e) dan lukisan tradisional Tiongkok dari berbagai periode dan seniman.


Balancing kelas — distribusi seimbang antara Jepang dan Tiongkok
Resolusi standar — semua gambar di-resize ke ukuran seragam sebelum patch extraction
Augmentasi — flipping, color jitter, slight rotation untuk memperkaya training set
Train/val/test split stratifikasi per kelas
Beberapa gambar memiliki watermark — diperhitungkan dalam patch filtering

Jepang: Ukiyo-e & Sumi-e

Ukiyo-e: woodblock prints bergaya dekoratif dengan outline jelas dan warna datar. Sumi-e: lukisan tinta berbasis teknik sapuan kuas ekspresif.

Tiongkok: Traditional Painting

Gongbi (detail halus) dan Xieyi (ekspresif, mirip kaligrafi) — dengan penggunaan warna dan komposisi yang khas berbeda dari estetika Jepang.

Patch Strategy

Grid 4×4 menghasilkan 16 patch per gambar. Patch yang terlalu besar (solid background >80%) difilter sebelum memasuki pipeline.

UMAP Visualization

UMAP & PCA — Feature Space

UMAP (Uniform Manifold Approximation and Projection) digunakan untuk memvisualisasikan apakah ResNet50 feature vectors benar-benar memisahkan lukisan Jepang dan Tiongkok dalam ruang dimensi rendah.


Jika cluster kedua kelas terpisah dengan jelas di UMAP 2D, ini mengkonfirmasi bahwa fitur yang diekstrak ResNet50 memang membawa informasi diskriminatif yang dapat dimanfaatkan SVM.


UMAP dijalankan pada train set feature vectors (post-ResNet50)
Cluster Jepang dan Tiongkok terbentuk dengan separasi yang jelas
Outlier dan patch ambigu terlihat di batas antar cluster
PCA digunakan sebelum SVM training untuk mengurangi noise dimensi tinggi
Interpretasi UMAP
Cluster compact → fitur konsisten per kelas
Inter-cluster gap → SVM dapat boundary yang jelas
Outlier region → patch yang membutuhkan centroid voting sebagai tiebreaker
PCA Pre-processing

PCA mereduksi 2048-dim ResNet50 output ke n komponen yang menjelaskan 95% variance — mempercepat SVM training signifikan tanpa kehilangan informasi diskriminatif.
